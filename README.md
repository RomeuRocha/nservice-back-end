# N.Service v1.0(BETA)

O aplicativo foi um trabalho academico apresentado a UNIRIOS(Centro Universitário do Rio São Francisco) em 2020 pelos alunos : Romeu Rocha e Anderson Alves,
STACKS
Front-End : JavaScript, React, React-admin, Axios...
